## English

### Codex doc management — 4 sub-tabs (Agents / Memories / Skills / MCP)

The Codex page is rebuilt: the legacy marker-managed-block flow is replaced by 4 sub-tabs, each editing the matching codex resource in raw mode with independent history keyed by SHA-256 path hash.

- Agents: raw read/write `AGENTS.md` at any path, Tauri file picker + paste-absolute-path; `.git/` detection auto-classifies project-root vs subdir; defaults to `~/.codex/AGENTS.md`
- Memories: fixed two entries `~/.codex/memories/MEMORY.md` (primary index) + `memory_summary.md` (auto summary) — the only user-editable indexes codex injects into the prompt; `raw_memories.md` / `rollout_summaries/` and other Phase 1-2 artifacts are intentionally not exposed
- Skills: scans `~/.codex/skills/<name>/SKILL.md` for raw editing; "Open folder" shells out to `open` / `xdg-open` / `explorer` so users edit non-SKILL.md companion files in Finder / Explorer
- MCP: `toml_edit::DocumentMut` round-trips `~/.codex/config.toml`, modifies only the `[mcp_servers.*]` section, preserves user's other sections + comments + key order; frontend JSON editing; `upsert_server` preserves unmodeled fields (`tools` per-tool approval / `env_vars` / future codex schema additions) instead of silently dropping them; Plugins sub-tab scans `~/.codex/plugins/cache/` with enable toggle + uninstall; Marketplace + Deeplink (`codex-app-transfer://` URL scheme + confirmation modal) wired on the backend, frontend entries hidden

Devin pre-merge safety / correctness fixes: tarball download 60s timeout + Content-Length pre-check + streaming size cap against OOM; plugin name / marketplace / version full-string rejection of `.` / `..` for path-safety; restore paths use atomic tmp+rename; `InstallInput` `#[serde(rename_all = "camelCase")]`; tarball wrapper collision false-positive, etc.

### Verification

- `cargo fmt --check` / `cargo check -p codex-app-transfer`
- 12 production MCP servers round-trip-edited without field loss
- 22+ skills, SKILL.md raw editing + "Reveal in Finder" verified on macOS
